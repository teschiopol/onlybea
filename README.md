# Vue 3 + TypeScript + Vite + PrimeVue 3 + PrimeFlex(Tailwind)

## Requirements

- node v20+

```bash
npm i
```

## Usage

For development

```bash
npm run dev
```

For production
```bash
npm run build
npm run preview
```
